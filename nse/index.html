NSE SCRIPT:


description=[[script for open port scanning]]
author = "thulasi doss"
portrule = function(host, port)
return port.protocol == "tcp" and port.state == "open"
end
action = function(host, port)
local sock = nmap.new_socket()
local result = sock:connect(host, port)
sock:close()
if result then
return "the\t".. port.number .."\tport is open"
else
return "the\t".. port.number .."\tport is close"
end
end


USAGE:


The above NSE script is use to display the available port and It will show the status for given port is open.


REQUIRMENT:


NMAP
KALILINUX


SETUP:


Copy the above code in to kali and save as luaport.nse

Move the nse file to /usr/share/nmap/script/


SAMPLE-OUTPUT:


CODE:


sudo nmap --script=luaport.nse --target ip--
